1)Que es un sistema de control de versiones?

Es un sistema que permite tener el registro de todas las modificaciones que has hecho a algún archivo, documento, etc, esto mismo ayuda a evitar perdidas de información y a su vez permite poder que uno se devuelva a las versiones anteriores de algún archivo antes de su ultima modificación, este mismo si se utiliza de forma remota facilita en gran medida el trabajo en equipo debido a que este organiza la información que se envía y renueva gracias a la identificación de quien hizo el cambio y que fue lo que cambio.
 
2)Que diferencia hay entre repositorio local y remoto?

El repositorio local es el computador en el que yo manejo los archivos por lo que solo funciona en ese computador, el remoto lo que permite es que cualquiera que forme parte de los modificadores puede hacer cambios en el repositorio desde cualquier lugar ósea desde otros computadores, ósea básicamente el local se limita a un solo equipo y el remoto puede usarce desde cualquier equipo.

3)Mencione tres diferencias entre Git y GitHub.

Uno es un sofware y el otro es un servicio.
Git se instala en el sistema y GitHub esta en la web.
Git es una herramienta de line a de comando y github proporciona una interfaz grafica.
